SnapSolve AI
SnapSolve AI is a full-stack, AI-powered study platform designed to help students solve academic problems faster and understand them better. Whether by typing a question or uploading an image, SnapSolve AI provides accurate answers, step-by-step explanations, and simplified breakdowns across multiple subjects.

Features
 AI-Powered Solutions
Get instant answers to academic questions using advanced AI models.
Image-to-Text (OCR) Support
Upload images of questions and extract text seamlessly for solving.

Step-by-Step Explanations
Understand not just the answer, but how to get there.
Simplified Breakdown
Complex solutions are broken down into easy-to-understand steps.
JAMB Preparation Mode
Specialized mode tailored for JAMB exam-style questions and explanations.
 Multi-Subject Support
Covers subjects like Mathematics, Physics, Chemistry, Biology, English, and more.

⚙️ How It Works
Input Question
Type your question OR upload an image.
Processing
OCR extracts text (if image is used).
AI model analyzes the question.
Output
Correct answer
Step-by-step explanation
Simplified breakdown for better understanding

📦 Installation
Clone the repository
Bash
git clone https://github.com/your-username/snapsolve-ai.git
cd snapsolve-ai
Install dependencies
Frontend
Bash
cd client
npm install
npm run dev
Backend
Bash
cd server


# Backend
npm run start


# Frontend
npm run dev
🔐 Environment Variables
Create a .env file in your backend directory and add:

AI_API_KEY=mine
OCR_API_KEY=mine
DATABASE_URL=mine
